Test Plans

PCBA testing:

  • check continuity off all nets, and ensure no nets are shorted
  • inspect all components to ensure they are properly seated

PCBA, Nucleo, and CAN testing:

  • Flash all connecting nucleos with testing firmware
  • attach PCBA shieds to all nucleos
  • connect all nucleos to the canbus
  • Run testing firmware:
    • Nucleo will start with both 1st floor LED's iluminated
    • When coresponding button is pressed, LED's will turn
      off and next floors LED's will turn on
    • 3rd floor LED's will cycle back to 1st floor
    • When each button is pressed a signal is sent on the
      canbus triggering any connected nucleos to change
      if the button matched the lit LED
